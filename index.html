Integração Notion

Passo 1 — Criar a integração
Acesse notion.com/my-integrations no navegador
Clique em + New integration
Dê um nome (ex: "Meus Exames") e selecione seu workspace
Clique em Save
Vá na aba Configuration e copie o Internal Integration Secret
O token começa com ntn_ (formato atual desde set/2024)
Cole esse valor no campo API Key acima

Passo 2 — Dar acesso à base de dados
Abra sua base "Meus Exames" no Notion
Clique nos ··· no canto superior direito
Role até o final e clique em + Add connections
Pesquise o nome da integração e selecione
Confirme o pop-up de permissão
⚠️ Sem esse passo a sincronização retorna erro 403

Passo 3 — Pegar o ID da base
Com a base aberta no navegador, a URL tem este formato:
notion.so/Nome-do-workspace/xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx?v=...
Copie os 32 caracteres entre a última / e o ?
Cole no campo ID da base acima — com ou sem hífens funciona

Estrutura obrigatória da base
Exame — Título (código, ex: VIT-D)
Valor — Número
Data — Data
Faixa — Texto (opcional)
